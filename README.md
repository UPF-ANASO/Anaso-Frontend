## 💨 Code Convention

- 변수 명 / 클래스 명 / 메소드 명은 기본적으로 카멜 케이스를 사용 한다.
  ex) getName(), User()...

- REST API 설계법 준수 (Get, Post, Put, Fetch, Delete)
- 함수는 1개의 기능 단위로 작성
- styled-component 이용 ( CSS5/SCSS )
- github PR요청 시

ex) [FE] 개발:: 로그인 Layout 구현 (민석)
[FE] 수정, 디버깅 :: 로그인 Layout (율희)
